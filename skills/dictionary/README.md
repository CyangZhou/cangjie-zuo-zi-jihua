# Dictionary Skill (字典)

这是一个元技能，用于构建和管理“仓颉造字”体系下的单字技能。

## 功能列表

- **定义 (Define)**: 什么是单字技能？如何造字？
- **组合 (Compose)**: 如何将单字串联成工作流？
- **检索 (Retrieve)**: 如何高效查找现有技能？
- **成语 (Idioms)**: 如何定义高级工作流？

## 快速开始

### 1. 查看造字模板
```bash
python .trae/skills/dictionary/creation/create_char.py
```

### 2. 搜索现有技能
```bash
python .trae/skills/dictionary/retrieval/search_chars.py
```

## 目录说明

所有单字技能应存放在 `characters/` 目录下，每个技能一个独立文件夹，包含 `SKILL.md`。
